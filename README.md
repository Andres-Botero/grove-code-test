# UI/Front-End Code Test for Grove

## 1. Installation

### Requirements
- Node.js >= 6

```shell
git clone https://github.com/Andres-Botero/grove-code-test.git app
cd app
npm install
npm run build
```

## 2. Run Application

```shell
npm start
```

## 3. Configuration

src/config.js
